# ADT
Bunch of abstract data types in C++. They (will) include pointer-based implementation of the following:
- Linked List - complete
- Stack - complete
- Queue - complete
- Binary Search Tree - complete
- Heap
  - array based - complete
  - ptr based - incomplete
- Priority Queue  
- AVL Tree
- 2-3 Tree
- 2-3-4 Tree
- Red-Black Tree
- Table  
- Hash Table
- Graph
